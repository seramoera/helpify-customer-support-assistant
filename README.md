# AI-Based Customer Support Assistant

This project develops an AI-powered customer support assistant using:

- Natural Language Processing (NLP)
- Convolutional Neural Networks (CNN) for intent classification
- Reinforcement Learning (RL) for response selection

## Dataset
Customer Support on Twitter Dataset
https://www.kaggle.com/datasets/thoughtvector/customer-support-on-twitter

## Features
- NLP preprocessing pipeline
- CNN intent classification
- Reinforcement learning response optimization
- Response template database
- CLI/Web interface for testing

## Project Structure
data/ – datasets  
src/ – core ML code  
models/ – trained models  
notebooks/ – experimentation
