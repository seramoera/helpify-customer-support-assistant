Features Added:
- token_count: Number of tokens after preprocessing
- char_count: Total character count
- contains_question_mark: Boolean flag for questions
- contains_exclamation: Boolean flag for exclamations
- sentiment_score: VADER compound sentiment (-1 to +1)
- has_number: Boolean flag for numeric content
